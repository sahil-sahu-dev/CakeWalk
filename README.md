# CakeWalk

## Features 💡 
* Uses HealthKit to get number of steps completed. 👟
* Allow multiple users to compare steps using firebase. 👩‍👩‍👦‍👦🔥
* Login with google

## Technologies used
* Firebase
* HealthKit
* Swift and SwiftUI
